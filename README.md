# react-native-zalo-kit

Zalo SDK implementation for React Native

## Installation

```sh
npm install react-native-zalo-kit
```

## Usage

```js
import { multiply } from "react-native-zalo-kit";

// ...

const result = await multiply(3, 7);
```
